# Capstone
A web app to explore Game of Thrones characters and houses

 🏰 Game of Thrones Explorer

📌 Project Description

This is a web application that allows users to explore characters and houses from the Game of Thrones universe using a public API.

 🚀 Features

* 🔍 Search characters by name
* 🏰 Filter by houses
* 🔃 Sort characters (name, culture)

 🌐 API Used
* https://anapioficeandfire.com/
 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

📷 Future Improvements

* Better UI design
* Pagination
